# N4BP2-ecDNA Analyzer

A machine learning tool to analyze the role of N4BP2 gene in extrachromosomal DNA (ecDNA) formation using Random Forest and SHAP.

## Features
- Generate synthetic gene expression data
- Train a Random Forest classifier to predict ecDNA presence
- Identify key genes (focus on N4BP2)
- Visualize results with SHAP summary plots

## Requirements
- numpy
- pandas
- scikit-learn
- shap
- matplotlib

## Usage
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run the main script: python main.py

## Results
Output plots are saved in the results/ folder.
